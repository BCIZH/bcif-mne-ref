# MNE-Python → Rust 依赖对照表

| 功能模块 | Python 对应 (SciPy/sklearn/MNE) | Rust 最终选型 | 实现逻辑与选型作用 |
|---------|--------------------------------|--------------|-------------------|
| 数据容器 | `numpy.ndarray` | `ndarray` | 核心多维数组容器，负责信号采样数据的存储、切片与通道管理。 |
| 实数频谱分析 | `scipy.fft.rfft` / `irfft` | `realfft` | 实数优化：专门针对电压等实数信号优化，性能优于通用复数 FFT。 |
| 复数频谱分析 | `scipy.fft.fft` / `ifft` | `rustfft` | 底层引擎：处理复数数据或解析信号（如 Hilbert 变换）时使用。 |
| 独立成分分析 | `sklearn.FastICA` | `petal-decomposition` | ICA 核心：提供 FastICA 算法实现，用于信号解混与去噪。 |
| 机器学习框架 | `sklearn` (其他部分) | `linfa` | 全能框架：处理聚类、回归、分类及模型评估（如逻辑回归、SVM）。 |
| 线代加速 | libopenblas / MKL | `ndarray-linalg` | 性能对标：绑定 OpenBLAS 等后端，加速 ICA 及 linfa 内部的矩阵运算。 |
| 核心滤波 | `scipy.signal.butter` | `idsp` | 信号处理：设计并执行 IIR 滤波（Butterworth）及窗函数生成。 |
| 采样率对齐 | `scipy.signal.resample` | `rubato` | 高质量重采样：采用 Sinc 插值防止传感器数据在频率转换时失真。 |
| 频率轴生成 | `scipy.fft.rfftfreq` | `ndarray` + 手动逻辑 | 公式实现：`f = [0..n/2] × fs/n`（Rust 通常手动向量化实现）。 |
| 频谱位移 | `scipy.fft.ifftshift` | `ndarray::slice` | 切片旋转：通过数组切片与数据块重组模拟 Python 的 shift 效果。 |
